# Extended-analysis-for-infer-subc-v2.0.0b1
This repository contains extended codes for:
- hierarchical clustering analysis
- dynamic time warping (DTW)
- extended organelle contacts analysis

(Based on infer-subc-v2.0.0b1 imaging data outputs).
These codes were used to generate data in the following preprint [doi: https://doi.org/10.64898/2026.02.10.704675]
